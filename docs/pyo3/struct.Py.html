#[repr(transparent)]pub struct Py<T>(_, _);

A Python object of known type.

Accessing this object is thread-safe, since any access to its API requires a Python<'py> GIL token.

See the guide for an explanation of the different Python object types.

Technically, it is a safe wrapper around NonNull<ffi::PyObject> with specified type information.
