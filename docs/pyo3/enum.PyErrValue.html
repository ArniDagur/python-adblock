pub enum PyErrValue {
    None,
    Value(PyObject),
    ToArgs(Box<dyn PyErrArguments>),
    ToObject(Box<dyn ToPyObject>),
}

Represents a PyErr value.

Caution:

When you construct an instance of PyErrValue, we highly recommend to use from_err_args method. If you want to to construct PyErrValue::ToArgs directly, please do not forget to call Python::acquire_gil.
